About Kordle

Inspiration

Kordle is inspired by LoLdle and Wordle, a game where you have to guess the champion or a word. The game is built using the Django REST Framework for the backend, while the frontend is made with HTML, CSS, and JavaScript.

How to Play

Enter a idol's name in the text box and click Guess to submit your answer. The game provides visual feedback through color-coded tiles:

Green: The category is correct.

Red: The category is incorrect.

Use your Kpop knowledge to try to make all the tiles turn green!

Scoring

You earn 1 point for each correctly guessed idol.

Reloading the page resets the champion to a new random selection.
